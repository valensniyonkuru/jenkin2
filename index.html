What is CI?

Continuous Integration (CI) is the practice of automatically building and testing code every time changes are pushed to a repository.

  • Developers push code to GitHub
  • Automated build runs
  • Automated tests run

What is CD?

Continuous Delivery/Deployment (CD) automatically prepares or deploys code to production after CI passes.

  • Deploy to staging
  • Run integration tests
  • Deploy to production
